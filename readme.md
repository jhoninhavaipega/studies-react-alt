## Studies React + Altjs

Studies to understand the operation of a front-end architecture, using emerging technologies.

### Technologies used
- React
- Altjs
- Babel
- Webpack
- NPM
- ES6
- SASS

### Application directory
```
./src/js
├── actions
│   └── NotificationActions.js
├── alt.js
├── app.js
├── components
│   ├── badge
│   ├── dropdown
│   ├── listNotifications
│   └── notifications
├── i18n
├── sources
│   └── NotificationSource.js
└── stores
    └── NotificationStore.js
```

### Running the project
- `npm run setup`
- `npm run server`
- open the url `http://localhost:8080/webpack-dev-server/` in your browser.
